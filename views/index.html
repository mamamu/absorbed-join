URL shortener Microservice using MongoDb

Pass a URL like this (don't forget to include /new/ before your url!):

https://absorbed-join.glitch.me/new/https://www.goodreads.com/book/show/18669431-node-js-the-right-way

Get json response with the original url and a shortened url like this:

{"url":"www.goodreads.com/book/show/18669431-node-js-the-right-way","shorturl":"https://absorbed-join.glitch.me/3"}

Paste shorturl into address bar for redirection goodness. Enjoy!
